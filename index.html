Audio Vaporwave Visualizer

I created this audio visualiser over a span of two weeks around mid-July of 2020, researching design and computer concepts such as Perlin Noise. With the Perlin Noise, I’ve made a terrain which bounces based on the amplitude, or volume, of the song it is paired with. In this specific example, I’ve chosen the song, “Ouke,” by Paco Amoroso and Ca7riel, and if one presses the play button, one can see that the moon, “pink ocean-like-net,” and the stars move relative to the amplitude/volume. The style was based off of the Vaporware/Vaporwave aesthetic which uses bold colors! I utilised the p5.js online editor and played around. I did this project alone, using javascript to code it all. This item is significant because it shows my artistic ability and will to problem solve, connecting music, design, and javascript to create something cool and fun!

Dance Vision

Using the posenet.js library, I used the p5.js editor to create a machine learning project which detects certain postures of the Bharatanatyam Indian Classical Dance Forms. I started fiddling with this app around January of 2020, but the project finished in April. For example, aramandi, or half-sitting position, is perfect when there is a certain angle created from the legs, bent at the knee. The resulting shape should be a rhombus; this app detects whether or not the perfect aramandi is achieved while dancing! This project was also done in javascript. Dance Vision is significant because I mixed two things I love, dance and technology, to generate something applicable in real life. I tutor dance students, and this app has come in handy when they try to self-correct at home!
